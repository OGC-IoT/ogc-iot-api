SensorThings API

The OGC IoT RESTful API is an OGC candidate standard for providing an open and unified way to interconnect IoT devices, data, and applications over the Web. IoT RESTful API is an open standard, builds on Web protocols and applies an easy-to-use RESTful style. The result is to provide a uniform way to expose the full potential of the Internet of Things.
